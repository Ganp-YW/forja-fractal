# Forja Fractal 🌌

**Forja Fractal** es una academia digital de alto rendimiento especializada en ciencias exactas y tecnología. Nuestro propósito es transformar la manera en la que aprendes Matemática, Física, Desarrollo Web y Programación, convirtiendo conceptos complejos en conocimiento sumamente práctico y accesible.

## 🚀 Módulos y Cursos Disponibles

- **Matemática:** Aprende el lenguaje del universo, desde álgebra y trigonometría hasta cálculo diferencial e integral avanzado.
- **Física:** Comprende las leyes que rigen el mundo natural. Estudiaremos cinemática, dinámica, electromagnetismo y termodinámica con aplicaciones reales.
- **Desarrollo Web:** Diseña y construye las interfaces del futuro. Aprende las bases de HTML5, estilos avanzados con CSS3 y añade interactividad pura con JavaScript moderno.
- **Programación:** Conviértete en desarrollador. Aprende lógica algorítmica, estructuras de datos y lenguajes modernos para construir software desde cero.

## 🧠 Metodología Fractal

Nuestra metodología de estudio se divide en tres pasos iterativos:
1. **El Núcleo:** Comenzamos desde la base absoluta. Entendemos los conceptos fundamentales como piezas individuales.
2. **La Iteración:** Aplicamos el patrón. A través de la práctica constante y la repetición de ejercicios dinámicos.
3. **La Expansión:** Escalamos lo aprendido. Integramos los conceptos básicos en problemas de alta complejidad.

## 🛠️ Tecnologías Utilizadas
- **HTML5:** Estructura semántica pura.
- **CSS3:** Diseño responsivo, Dark Mode UI, Grid y Flexbox.
